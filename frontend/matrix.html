📊 ภาพรวมตาราง
1
ตารางนี้แสดง ความสอดคล้อง ระหว่างผู้มีส่วนได้ส่วนเสียของหลักสูตรกับ PLO สีม่วง และ MLO สีน้ำเงิน
2
กดที่ วงกลม ○ ในตารางเพื่อ Toggle ✓/○ — ระบบบันทึกลง Database ทันที แถบสถานะมุมขวาบนจะแจ้งผล
3
แถบด้านล่างตารางแสดง PLOs / MLOs Covered — จำนวน PLO และ MLO ที่มีการเชื่อมโยงอย่างน้อย 1 แถว
✏️ จัดการแถวข้อมูล
4
กดปุ่ม + เพิ่มแถวใหม่ ด้านล่างตาราง — กรอก กลุ่ม · ชื่อรายการ · คำอธิบาย แล้วกด บันทึก
5
Hover ที่แถวใด ๆ เพื่อให้ปุ่ม ✏️ แก้ไข และ 🗑 ลบ ปรากฏขึ้นทางขวาของชื่อแถว
6
การ เพิ่ม / แก้ไข / ลบ แถวจะยังไม่บันทึกจนกว่าจะกดปุ่ม บันทึก สีเขียวมุมขวาบน ยังไม่บันทึก
7
ใช้ Ctrl + Enter เพื่อบันทึก Modal แก้ไขแถวได้รวดเร็ว และ Esc เพื่อปิด Modal ทุกประเภท
📥 Import / Export Excel
8
กดปุ่ม Template เพื่อดาวน์โหลดไฟล์ Excel ตัวอย่าง — คอลัมน์จะตรงกับ PLO/MLO ที่มีในระบบปัจจุบัน
9
กรอกข้อมูลใน Template แล้วกด Import Excel — ใส่ 1 ในคอลัมน์ PLO/MLO ที่ต้องการ ✓ และ 0 หรือเว้นว่างสำหรับ ○
10
กดปุ่ม Export Excel เพื่อดาวน์โหลดตารางปัจจุบัน หรือ PDF เพื่อสั่งพิมพ์ผ่าน Print Dialog ไม่ต้องบันทึกก่อน Export
🌐 การเผยแพร่
11
Toggle "เผยแพร่" ในแถบสีเขียวด้านบนตาราง — เมื่อเปิด ข้อมูลจะแสดงในหน้าสาธารณะ bba-matrix1.html ทันที
12
กด Preview เพื่อเปิดหน้าสาธารณะในแท็บใหม่และตรวจสอบก่อนเผยแพร่จริง ปิดเผยแพร่ก่อนแก้ไขใหญ่
💡 เคล็ดลับ: แถบสถานะ "โหลดจาก Database แล้ว" (สีเขียว) หมายความว่าเชื่อมต่อสำเร็จ — หากแสดง "เชื่อมต่อไม่ได้" (สีแดง) ให้ตรวจสอบ Server และ fms-api-client.js ก่อนบันทึก
