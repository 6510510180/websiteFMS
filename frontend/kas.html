หน้าตารางวิเคราะห์ความสอดคล้อง KAS (kas.html)
📋 ภาพรวมหน้านี้
1
หน้านี้ใช้ เชื่อมโยง PLO / MLO / CLO เข้ากับองค์ประกอบ KAS — แบ่งเป็น 3 Tab ได้แก่ PLO, MLO และ CLO
2
KAS ย่อมาจาก 3 มิติหลักของผลลัพธ์การเรียนรู้:
📖 K — Knowledge ความรู้ ♥ A — Attitude ทัศนคติ ⚙ S — Skill ทักษะ
ข้อควรรู้: ข้อมูล PLO และ MLO ในหน้านี้ดึงมาจาก Database โดยตรง — ต้องบันทึกข้อมูลที่ หน้าจัดการ PLO/MLO ก่อน จึงจะแสดงผลได้ถูกต้อง
🚀 ขั้นตอนเริ่มต้นใช้งาน
1. ตรวจสอบ Banner สถานะ → 2. กด "รีโหลด PLO/MLO" → 3. Map KAS แต่ละรายการ → 4. กด "บันทึก KAS"
3
Banner สถานะ แสดงผลการโหลดข้อมูล PLO/MLO จาก Database:
● สีเขียว — โหลดสำเร็จ พร้อมใช้งาน
● สีเหลือง — ไม่พบข้อมูล กรุณาไปบันทึกที่หน้า PLO/MLO ก่อน
4
กดปุ่ม รีโหลด PLO/MLO มุมขวาของ Banner เพื่อดึงข้อมูลใหม่จาก Database เมื่อต้องการซิงค์ข้อมูลล่าสุด
🔗 การ Map KAS ให้ PLO และ MLO
5
เลือก Tab PLO หรือ MLO — แต่ละแถวแสดงรหัส, คำอธิบาย และปุ่มแก้ไขแยกตาม K / A / S
6
กดปุ่ม แก้ไข K / แก้ไข A / แก้ไข S ในแต่ละแถวเพื่อเปิด Dialog เลือกรายการ KAS ที่สอดคล้อง — สามารถเลือกได้หลายรายการต่อครั้ง
7
Tab MLO มี Sidebar ซ้ายมือสำหรับเลือก กลุ่มวิชาเอก — คลิกชื่อกลุ่มเพื่อสลับแสดง MLO ของกลุ่มนั้น
📚 การ Map PLO/MLO ให้ CLO
8
Tab CLO ใช้เชื่อมโยง Course Learning Outcomes กับ PLO/MLO — เลือกผ่านแผงด้านขวาตามลำดับ:
เลือกหลักสูตร → เลือก Major → เลือกรายวิชา → คลิก CLO เพื่อ Map
9
คลิกที่ช่อง Linked PLO/MLO ของแต่ละ CLO เพื่อเปิด Dialog — เลือก PLO และ MLO ที่สอดคล้องแล้วกด Save Changes
10
ระบบจะพยายามบันทึก CLO mapping ไปยัง API อัตโนมัติ — หาก API ไม่ตอบสนอง ข้อมูลจะถูกเก็บ ชั่วคราวในเครื่อง และแสดง Toast สีเหลืองแจ้งเตือน ⚠ ตรวจสอบด้วย
📂 Import / Export ข้อมูล KAS
ฟังก์ชัน รายละเอียด รูปแบบไฟล์
Template ดาวน์โหลดไฟล์ตัวอย่างพร้อม header PLO/MLO และ KAS columns .xlsx (2 Sheet: PLO_KAS, MLO_KAS)
Import Excel อัปโหลดไฟล์ที่กรอก ✓ หรือ 1 ในช่อง K/A/S ที่ต้องการ map .xlsx / .xls
Export Excel ส่งออก KAS Mapping ทั้งหมดพร้อมรหัสที่เลือก .xlsx (2 Sheet: PLO_KAS, MLO_KAS)
Export PDF เลือก PLO / MLO / CLO หรือทั้งหมด — เปิด Print Dialog PDF ผ่าน Browser Print
11
กด Template ก่อนเสมอ เพื่อให้ได้ไฟล์ที่มี header ถูกต้อง — จากนั้นกรอก ✓ หรือ 1 ในช่อง K/A/S ที่ต้องการ แล้วกด Import
12
Import เพิ่มต่อท้าย ข้อมูลที่มีอยู่ (Append Mode) — ไม่ได้แทนที่ข้อมูลเดิม Append Mode
13
Export PDF มีให้เลือก 4 แบบ: ตาราง PLO, ตาราง MLO, ตาราง CLO, หรือ ทั้งหมด — คลิกลูกศรข้างปุ่มเพื่อเลือก
💾 การบันทึก KAS Mapping
14
กดปุ่ม บันทึก KAS (สีเขียว) มุมขวาบนเพื่อบันทึก KAS Mapping ของ PLO และ MLO — ข้อมูลจะถูกเก็บไว้ใน Browser บันทึกใน Browser เท่านั้น
15
CLO Mapping บันทึกผ่าน API อัตโนมัติ ทันทีที่กด Save Changes ใน Dialog — ไม่ต้องกดปุ่ม "บันทึก KAS" อีกครั้ง
16
หากปิด Browser โดยไม่บันทึก PLO/MLO KAS — ข้อมูลที่ยังไม่ได้บันทึกจะ สูญหาย ควรบันทึกก่อนปิดเสมอ
⚠ Admin Note: KAS Mapping ของ PLO/MLO ถูกเก็บใน LocalStorage ของ Browser เท่านั้น — หากเปลี่ยนเครื่องหรือล้าง Browser Cache ข้อมูลจะหาย กรุณา Export Excel สำรองไว้ เป็นประจำ
⚙️ การจัดการ KAS Master Data
17
รายการ K / A / S ในระบบดึงมาจาก API /api/kas — หากต้องการเพิ่ม/แก้ไขรายการ KAS ต้องดำเนินการที่ฐานข้อมูลหลักโดย Admin
18
ช่อง Search ใน Dialog ช่วย กรองรายการ KAS แบบ Real-time — พิมพ์รหัสหรือคำค้นเพื่อหารายการที่ต้องการได้รวดเร็ว
💡 แนวทางสำหรับ Admin: ควร Map KAS ให้ PLO ก่อน → จากนั้น MLO → สุดท้าย CLO เพื่อให้ข้อมูลสอดคล้องกันในทุกระดับ และ Export Excel สำรองไว้ทุกครั้งหลังอัปเดตข้อมูล
